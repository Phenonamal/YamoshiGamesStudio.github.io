Anime Legends

Anime Legends is a Turn Based Anime Gacha Game with cubed like models. The main inspiration for this game was Dragon Ball Legends, and other gacha like games. The game currently consists of 6 characters which can be summoned using the in game currency "gems", collecting multiple instances of the same character, would "stack it", which gives the character a star, each star boosts the damage of the character. Each character can block, basic attack, or use 4 different skills. For each turn you first decide what you'd like your char to do and afterwards, once both players are done with their turn. The battle sequence will start, in which all the attacks you've stacked up will perform in turns.

Current Project (Name is a Work In Progress)

This project is basically the same as the first project, however this time it's a trading card game. This is a online trading card game using anime characters as it's characters.
