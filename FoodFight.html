Food Fight

h1>

Food Fight is a gmae I've made with a team in a week for a jam, the theme for this jam was tags. So the purpose of this jam is basically lasting as long as possible as it's essentially waves that you will have to defeat. Each waves more formidable than the last with new enemies popping up and more enemies in total that you got to defeat. Use your tags to return the mutated food back to their original state! Have some fun with this little game I've created!
